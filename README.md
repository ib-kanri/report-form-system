## 🔧 カスタマイズ

### スプレッドシートID変更
`Code.gs` の1行目を編集:
```javascript
const SPREADSHEET_ID = 'YOUR_NEW_SPREADSHEET_ID';
```

### Apps Script URL変更
すべてのHTMLファイル内の以下の行を検索・置換:
```javascript
const SCRIPT_URL = 'YOUR_NEW_APPS_SCRIPT_URL';
```

### デザインカスタマイズ
各HTMLファイルの `<style>` タグ内の CSS変数を編集:
```css
:root {
    --primary: #YOUR_COLOR;
    /* ... */
}
```
